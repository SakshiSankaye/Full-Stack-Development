Important HTML Tags with Syntax and Description

HTML Tag Syntax Description
<html> <html>...</html> Root element of an HTML document
<head> <head>...</head> Contains metadata of the webpage
<title> <title>Page Title</title> Defines the title of the webpage
<body> <body>...</body> Contains visible webpage content
<h1> <h1>Heading</h1> Main heading of the page
<p> <p>Paragraph</p> Defines a paragraph
<br> <br> Inserts a line break
<hr> <hr> Creates a horizontal line
<a> <a href="URL">Link</a> Creates a hyperlink
<img> <img src="image.jpg" alt="image"> Displays an image
<ul> <ul>...</ul> Defines an unordered list
<ol> <ol>...</ol> Defines an ordered list
<li> <li>Item</li> Defines a list item
<table> <table>...</table> Creates a table
<tr> <tr>...</tr> Defines a table row
<th> <th>Heading</th> Defines a table header
<td> <td>Data</td> Defines a table data cell
<form> <form>...</form> Creates an HTML form
<input> <input type="text"> Accepts user input
<div> <div>...</div> Defines a block-level container
