# Assignment 5: Portfolio Website with MongoDB Integration

## Problem Statement

Develop a dynamic portfolio website using Node.js and Express, and connect it with a NoSQL database (MongoDB) to store and manage data.

## Objective
To build a full stack web application
To integrate frontend with backend using Node.js and Express
To connect and perform operations using MongoDB
To manage dynamic data such as contact form or user details

## Technologies Used
HTML
CSS
JavaScript
Node.js
Express.js
MongoDB

## Description

This project demonstrates the development of a dynamic portfolio website integrated with a backend server and database.

The frontend is developed using HTML, CSS, and JavaScript to display personal information, projects, and contact details in a structured and visually appealing manner.

The backend is built using Node.js and Express.js, which handles routing, server-side logic, and communication between the frontend and database.

MongoDB is used as a NoSQL database to store and retrieve user data such as contact form submissions. This integration enables dynamic interaction between frontend and backend, making the application fully functional.

## Features
Personal portfolio interface
Backend server using Express.js
MongoDB database integration
Contact form data storage
Data retrieval from database
Organized project structure (client + server)
Responsive and clean UI
 
## Steps to Run the Project
1. Install Dependencies
npm install
2. Setup MongoDB
Install MongoDB locally OR use MongoDB Atlas
Create a database and collection
Update the MongoDB connection string in .env file
3. Run the Server
node server.js
OR
nodemon server.js
